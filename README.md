# NEW_Luigi-s_casino

Versione base in MVC del gioco di carte di Luigi's Casino (Super Mario), giocabile da terminale.

## Avvio rapido

```bash
javac *.java
java Controller
```

## Struttura MVC

- `Model.java`: logica del mazzo, round e punteggio.
- `Viewer.java`: input/output da console.
- `Controller.java`: coordina il flusso di gioco.
